# Networking Basics

## What is a Network?
A network is a group of devices that communicate with each other.

## Important Concepts

### IP Address
An IP address identifies a device on a network.

### DNS
DNS converts domain names into IP addresses.

### HTTP vs HTTPS
- HTTP = unencrypted
- HTTPS = encrypted

## What I Learned
- Every device has an IP address
- DNS acts like the internet's phone book
- HTTPS protects data in transit
